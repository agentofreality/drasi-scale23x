# Drasi: A New Take on Change-Driven Architectures

Presentation materials for [SCaLE 23x](https://www.socallinuxexpo.org/scale/23x) — a session introducing [Drasi](https://drasi.io), an open-source Data Change Processing platform (CNCF Sandbox), and the change-driven architecture paradigm.

## What's Inside

```
outline/        # Talk outline and structure notes
presentation/   # Reveal.js slide deck
start.sh        # One-command launcher for the presentation
```

## Running the Presentation

**Prerequisites:** [Node.js](https://nodejs.org/) (v18+)

```bash
./start.sh
```

This installs dependencies (if needed) and serves the presentation at **http://localhost:3000**.

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `S` | Open speaker notes |
| `→` / `↓` | Next slide |
| `←` / `↑` | Previous slide |
| `Esc` | Slide overview |
| `Home` (`Fn + ←` on Mac) | Jump to first slide |
| `F` | Fullscreen |
